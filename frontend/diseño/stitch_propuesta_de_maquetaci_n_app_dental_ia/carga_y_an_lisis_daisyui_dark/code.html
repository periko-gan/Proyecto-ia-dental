Protocol Guide

Clinical Visual Guide

Ensure high-fidelity analysis by following these imaging protocols.

  • Optimal Lighting
    Soft Neutral Light

    Avoid direct flash on shiny surfaces. Use secondary, diffused white light sources.

  • Correct Angle
    Orthogonal View

    Capture images perpendicular to the tooth surface. Maintain horizontal occlusal plane.

  • Focus Calibration
    center_focus_weak

    The AI requires min 300 DPI. Use optical focus lock.
